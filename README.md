# WebScraper
Web Scraper created using Node and Nightmare

### Get Data of NFL Teams and Players using Node and Nightmare

### Instruction 

#### First Clone this repository

```
git clone 
```
#### Install node modules

```
npm install 
```

* [EJS - JS Template Engine](https://ejs.co/)
